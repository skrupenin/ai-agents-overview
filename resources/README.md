# AI Agents Resources & Learning Materials

## Academic Papers & Research

### Foundational Papers
1. **"Attention Is All You Need"** (Vaswani et al., 2017)
   - Introduced Transformers, foundation of modern LLMs
   - [Link](https://arxiv.org/abs/1706.03762)

2. **"BERT: Pre-training of Deep Bidirectional Transformers"** (Devlin et al., 2018)
   - Breakthrough in NLU
   - [Link](https://arxiv.org/abs/1810.04805)

3. **"ReAct: Synergizing Reasoning and Acting in Language Models"** (Yao et al., 2023)
   - Agent reasoning pattern
   - [Link](https://arxiv.org/abs/2210.03629)

4. **"Retrieval-Augmented Generation"** (Lewis et al., 2020)
   - RAG pattern for knowledge grounding
   - [Link](https://arxiv.org/abs/2005.11401)

### Multi-Agent Systems
1. **"Multi-Agent Reinforcement Learning: A Selective Overview"** (Zhang et al., 2019)
   - Comprehensive overview
   - [Link](https://arxiv.org/abs/1911.10635)

2. **"Emergent Complexity via Multi-Agent Competition"** (Bansal et al., 2017)
   - Competitive multi-agent learning
   - [Link](https://arxiv.org/abs/1710.03748)

## Next Steps for Learning

### Beginner Path (0-6 months)
1. Take intro course (DeepLearning.AI)
2. Build simple agent (chatbot)
3. Read case studies (this repo!)
4. Contribute to open-source agent project

### Intermediate Path (6-12 months)
1. Deep dive into one domain (NLP, RL, etc.)
2. Build production-grade agent
3. Read research papers
4. Experiment with multi-agent systems

### Advanced Path (12+ months)
1. Research novel agent architectures
2. Publish findings
3. Contribute to frameworks
4. Build company-specific agent solutions
